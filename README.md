## 技术栈

- 纯前端：HTML + CSS + JavaScript，单文件或少量文件
- 零依赖，浏览器直接打开
- 支持 GitHub Pages 部署

## 使用方式

1. 直接用浏览器打开对应的 `index.html`
2. 或部署到 GitHub Pages / 任何静态服务器

## 添加新工具

1. 在根目录创建新文件夹（如 `new-tool/`）
2. 放入 `index.html`
3. 在根目录 `index.html` 的工具列表中添加卡片
4. Push 即上线
